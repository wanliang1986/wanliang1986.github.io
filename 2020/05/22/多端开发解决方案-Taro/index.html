多端开发解决方案--Taro

发表于 2020-05-22 | 分类于 前端 |

Taro是一套遵循react语法规范的多端开发解决方案。
使用Taro，我们可以只书写一套代码，再通过 Taro 的编译工具，将源代码分别编译出可以在不同端（微信 / 京东 / 百度 / 支付宝 / 字节跳动 小程序、快应用、H5、React-Native 等）运行的代码。

安装与项目目录详解

Taro 项目基于 node，请确保已具备较新的 node 环境（>=8.0.0）

  • 全局安装@tarojs/cli：npm install -g @tarojs/cli
  • 使用命令创建模板项目：taro init 项目名
  • 进入项目目录，安装依赖：npm install
  • 项目目录：
  • 启动命令（h5）：npm run dev:h5

Taro react-hooks写法，路由及子组件

react-hooks写法

在项目生成的默认文件中，所用的是react的基础写法，现在react开发更喜欢用简洁的react-hooks写法，所以，我们对index.jsx进行改造。

  • 修改index.jsx中的引入，将component引入改为useState
    import Taro, { useState } from '@tarojs/taro'
  • 改造原本class组件的写法为react-hooks的function写法
  • 重新运行npm run dev:h5

    运行成功，说明react-hooks可以使用
